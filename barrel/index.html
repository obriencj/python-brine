reset()[source]¶

Clears the internal cache, meaning any future sets or gets from this Barrel will cause full brining or unbrining rather than returning an already computed value.

If you retrieved a value from this barrel and want to load a new copy (possibly with different globals), calling reset() is a way to achieve such.
