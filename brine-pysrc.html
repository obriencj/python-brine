 16  """ 
 17  Provides a simple way to pickle/unpickle function objects. 
 18   
 19  To truly pickle a function we need to be able to duplicate its code 
 20  and its closures. By default, pickle will simply store the function's 
 21  name, and then attempt to associate that with a function when 
 22  unpickling. This of course fails when the function is anonymous or not 
 23  otherwise defined at the top level. 
 24   
 25  In order to mark a function for actual storage, use the `brine` 
 26  function to create a BrineFunction instnace, which may then be 
 27  pickled. Later, after unpickling the BrineFunction, call `unbrine` to 
 28  get a new copy of the original function 
 29   
 30  See the brine.barrel module in order to pickle recursive functions, 
 31  mutually recursive functions, and the like. 
 32   
 33  @author: Christopher O'Brien  <obriencj@gmail.com>
